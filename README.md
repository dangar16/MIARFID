# RFA - Reconocimiento de Formas y Aprendizaje Automático (Pattern Recognition and Machine Learning)
Developed a multi-model pipeline (CNNs, MLPs, and Linear Models) to classify Alzheimer’s stages in brain imaging, including benchmarking against pre-trained architectures.

# SMA - Sistemas Multiagente (Multi-Agent Systems)
Implemented a multi-agent system to simulate differents types of auctions, including English, Dutch, and Vickrey auctions, with agents employing various bidding strategies.

# Tecnicas Metaheuristicas (Metaheuristic Techniques)
Applied metaheuristic algorithms (Genetic Algorithms, Simulated Annealing, and Particle Swarm Optimization) to solve combinatorial optimization problems, such as a variation of the Knapsack problem that had more constraints than the original problem.

# TLH - Tecnologias de Lenguajes Humanos (Human Language Technologies)
Developed from scratch a encoder architecture for a transformer-based model to analyze differents types of architechtures and comparing them to see which one performs better in a specific task, such as retrieving information from a large corpus of text data.

# Traducción Automática (Machine Translation)
- Applied different transformer models such as Llama 2, Mistral, NLLB to perform machine translation tasks, evaluating their performance on various language pairs and fine-tuning them for specific translation tasks to improve accuracy and fluency.

- Developed a pipeline for machine translation that includes transcribing audio input and translating it into the target language exploring ent to end machine translation systems and cascade systems, comparing their performance and efficiency.
