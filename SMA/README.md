# Multi Agent Systems

This repository contains the code for the Multi Agent Systems. The code is organized into different folders, each containing the code for a specific assignment or project.

This project involved the implementation of a multi-agent system to simulate a scenario where multiple agents with differents roles participate in an auction.

Fo each different type of auction, a different multi agent system was developed using ADK (Agent Development Kit) from Google.
Each folder contains the agent.py file which includes the implementation of the agents.
