# Metaheuristic Techniques

This repository contains implementations of various metaheuristic techniques for optimization problems. The techniques included are:
- Genetic Algorithms
- Simulated Annealing

For each technique, there are two different notebooks (final and test). The `test` notebooks contain all the experimentation of the technique exploring different parameters and configurations, while the `final` notebooks contain the best configuration found in the `test` notebooks and are used to generate the final results.
